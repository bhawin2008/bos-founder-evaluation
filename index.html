Flag Rules

Performance tracking guidelines

Red Flags

⚑ x1
Missed Deadline
If a task is not completed by its due date, the assignee automatically receives 1 red flag. This is detected automatically by the system.
⚑ x1
Below Expectation
If a task is completed but the result does not meet the expected standard, the assignee receives 1 red flag. Assigned during task completion review.
⚑ x3
Blunder
If a person makes a serious mistake or blunder during task progress, they receive 3 red flags. Can be reported at any time via the flag button on active tasks.

Green Flags

⚑ x1
Completed Perfectly
If a task is completed perfectly and meets all expectations, the assignee receives 1 green flag. Assigned during task completion review.
⚑ x2
Extraordinary Result
If a task is completed with an extraordinary result that exceeds expectations, the assignee receives 2 green flags. Assigned during task completion review.

Special Rules

♛ +1ea
Leadership Multiplier
If a person mentors 2+ teammates, helps in cross-team collaboration, or is referenced positively by others, they receive +1 green flag per qualifying criterion per month.
⚠ warn
Monthly Red Warning
If a member accumulates 2+ red flags in a single month (even if net score is still positive), a warning badge is triggered for immediate attention.
⇩ -50%
Red Flag Decay
If no red flags are received for 2 consecutive months, all previous red flags lose 50% weight. This rewards sustained improvement and gives members a path to recovery.

How It Works

1
Assign tasks to team members with a due date.
2
Overdue detection runs automatically. If a task passes its due date, 1 red flag is assigned to the member.
3
Report a blunder anytime using the flag button on completed/cancelled tasks. This assigns 3 red flags immediately.
4
Complete a task by clicking the checkbox. A review modal lets you rate the quality and assign green or red flags.
5
Leadership Multiplier — use the crown button on member cards to award green flags for mentoring, collaboration, or positive references.
6
Red Flag Decay — 2 clean months (no reds) automatically halves old red flag weight. Monthly Warning triggers when 2+ reds land in one month.
7
Track performance on the Dashboard. Member trends show Rising/Flat/Falling. Red Alerts fire for 2+ consecutive red-zone months.
