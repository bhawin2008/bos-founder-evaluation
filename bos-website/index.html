Team Operations

BossTeam

A lightweight team management dashboard for tracking members, tasks, roles, and performance. Built with vanilla JS and localStorage -- no backend required.

  • Dashboard with real-time team stats
  • Task management with priority and status tracking
  • Role-based team organization with color coding
  • Month-over-month analytics with trend visualization
  • Member performance tracking and comparison

DecisionOS

A structured decision-making framework that processes decisions through three analytical layers: Clarity, Truth, and Alignment. Get automated verdicts, not just opinions.

  • Three-layer analysis: Clarity, Truth, Alignment
  • 8-dimension weighted scoring system (100-point scale)
  • Automated verdicts: DO, DON'T, WAIT, or TEST
  • Red flag detection and CALM emotional state check
  • Experiment plan generation for uncertain decisions

Getting Started

Up and running in minutes.

No sign-ups. No credit cards. Just clone and go.

1

Clone the repo

Pull down the BOS repository. Everything you need is included.

git clone <repo-url>
2

Pick a tool

Open BossTeam directly in your browser, or install DecisionOS dependencies with npm.

cd decision-os && npm install
3

Start building

Manage your team, make structured decisions, and focus on what matters.

npm run dev

Stop winging it. Start operating.

BOS is free, open source, and built to solve real founder problems. No signup required.
